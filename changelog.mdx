## Upgrade Instructions

Follow these steps to upgrade Janek Unon safely.

<Steps>
  <Step title="Backup Data" icon="database">
    Export your projects:
    
    ```bash
    janek-unon export --all > backup.json
    ```
  </Step>
  
  <Step title="Install Latest" icon="download">
    Update via your package manager.
    
    <Tabs>
      <Tab title="npm" icon="package">
        ```bash
        npm uninstall janek-unon
        npm install janek-unon@latest
        ```
      </Tab>
      <Tab title="yarn" icon="package">
        ```bash
        yarn remove janek-unon
        yarn add janek-unon@latest
        ```
      </Tab>
    </Tabs>
  </Step>
  
  <Step title="Verify" icon="check-circle">
    Run tests:
    
    ```bash
    npm run test
    janek-unon verify --config config.json
    ```
  </Step>
</Steps>

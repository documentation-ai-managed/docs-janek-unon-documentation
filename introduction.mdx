## What is Janek Unon

Janek Unon provides a flexible documentation space where you organize and manage project docs. You create structured pages, embed code examples, and customize layouts to fit your needs. This default space supports MDX for rich content like interactive components and diagrams.

<Callout kind="info" collapsed="false">
  Janek Unon uses MDX, so you combine Markdown with JSX components for dynamic docs.
</Callout>

## Getting Started

Follow these steps to set up your first documentation site:

<Steps>
  <Step title="Install Dependencies" icon="download" title-type="p">
    Use your preferred package manager to install the core package.

    <CodeGroup show-lines="true" tabs={[]}>
      ```bash
      npm install janek-unon mdx-docs
      ```

      ```bash
      yarn add janek-unon mdx-docs
      ```

      ```bash
      pnpm add janek-unon mdx-docs
      ```
    </CodeGroup>
  </Step>

  <Step title="Configure Theme" icon="settings" title-type="p">
    Update your `next.config.js` or equivalent to set the brand color.

    ```javascript
    const config = {
      theme: {
        colors: {
          primary: '#3B82F6'
        }
      }
    };
    ```
  </Step>

  <Step title="Create Your First Page" icon="file-text" title-type="p">
    Add `introduction.mdx` in your docs folder and start writing.
  </Step>
</Steps>

## Documentation Management Overview

Manage your docs across platforms with these tailored approaches:

<Tabs>
  <Tab title="GitHub" icon="github">
    Host your repo on GitHub and deploy previews automatically.

    ```bash
    git init
    git add .
    git commit -m "Initial docs"
    git push origin main
    ```
  </Tab>

  <Tab title="Vercel" icon="zap">
    Connect your repo to Vercel for instant deployments.

    1. Log in to Vercel.

    2. Import your GitHub repo.

    3. Set build command to `npm run build`.
  </Tab>

  <Tab title="Netlify" icon="cloud">
    Drag your repo or connect via OAuth for continuous deployment.
  </Tab>
</Tabs>

## Brand and Customization

Customize Janek Unon to match your brand. Set the primary color to `#3B82F6` in your theme config for a blue accent.

<Expandable title="Advanced Theme Config" default-open="false">
  Override CSS variables for full control:

  ```css
  :root {
    --color-primary: #3B82F6;
    --color-primary-hover: #2563EB;
  }
  ```
</Expandable>

<Callout kind="tip" collapsed="false">
  Test customizations in development mode with `npm run dev` to preview changes live.
</Callout>
